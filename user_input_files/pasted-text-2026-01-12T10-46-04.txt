Talk to Your Accounts
A conversational offline-first desktop accounting application for small businesses, retailers, and traders. Talk to your accounts using natural voice or text in English, Hindi, or Hinglish.

Features
Core Capabilities
Voice-Driven Transactions: Record sales, purchases, and expenses by speaking naturally. Simply say "Sold goods worth ₹5,000 to Sharma Traders" and the system handles the rest.
Conversational Business Queries: Ask questions about your business in plain language. Examples include "How much did I sell today?", "Who hasn't paid me yet?", and "What's my profit this month?".
Multi-Language Support: Works seamlessly in English, Hindi, and Hinglish. The system understands mixed-language commands common in Indian business environments.
Intelligent Explanation Engine: Beyond showing numbers, the system explains what they mean for your business. It highlights trends, identifies problems, and suggests actions.
Offline-First Architecture: Complete accounting functionality without internet. All data stays securely on your device with local encryption.
Voice Report Generation Features
The application now includes comprehensive voice report generation capabilities that read reports aloud and provide smart business insights.

1. Voice Report Reading
The system can read financial reports aloud using text-to-speech technology, making it accessible for users who prefer audio feedback or have visual impairments.

Features:

Automatic narration of report summaries after generation
Toggle between visual and audio responses
Hinglish voice responses for better accessibility
Different voice patterns for different report types
How to Use:

Simply ask "Show me my sales report" or "Sales report chahiye"
The system generates the report and reads the key highlights aloud
Toggle the "Auto-read report" checkbox to enable or disable this feature
Example Commands:

"Show me sales report"
"Mujhe profit and loss report dikhao"
"Bikri ka report chahiye"
"GST report generate karo"

Installation
Prerequisites
Node.js 18 or higher
npm or yarn package manager
Setup Steps
1.
Clone or Navigate to the Project
bash
cd talk-to-your-accounts
1.
Install Dependencies
bash
npm install
1.
Start Development Mode
bash
npm run dev
This will start both the React development server and Electron simultaneously.

1.
Build for Production
bash
npm run build
npm run build:react
1.
Run Built Application
bash
npm start
Building Desktop Executables
To create platform-specific installers:

bash
# For Windows
npm run build -- --win

# For macOS
npm run build -- --mac

# For Linux
npm run build -- --linux
First-Time Setup
1.
Launch the Application: After running npm start, you'll see the welcome screen.
2.
Create Your Account: Enter your business name and a 4-digit PIN. This PIN protects all your data.
3.
Start Talking: The assistant will greet you and explain how to use voice commands.
Architecture
Technology Stack
Electron: Desktop application framework
React: User interface library

xlsx: Excel export functionality

Troubleshooting
Voice Recognition Not Working
1.
Check microphone permissions in system settings
2.
Ensure you're using Chrome or Edge browser
3.
Speak clearly and at moderate pace
4.
Try text input if voice continues to have issues
Report Generation Issues
1.
Ensure you have data for the selected period
2.
Check date range is valid
3.
Try regenerating the report
4.
Verify database connection
Export Problems
1.
Check if jsPDF and xlsx libraries are loaded
2.
Verify file permissions for download
3.
Try a different export format
Application Won't Start
1.
Delete node_modules and reinstall: rm -rf node_modules && npm install
2.
Check Node.js version: node --version (needs 18+)
3.
Check console for error messages
Data Recovery
1.
Locate backup file in user data directory
2.
Export via Settings > Data Management
3.
Reinstall application if needed
